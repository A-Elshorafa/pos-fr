# Merchant + POS Mini System (Frontend - Angular)

This project is a simplified Fintech-oriented Merchant & POS management system built using **Angular + TypeScript**.  
It demonstrates core concepts required in real-world payment and merchant platforms such as:

- Merchant Dashboard
- POS Device Management
- Payment Creation
- Transaction History
- Authentication (mock)
- Event-driven UI updates

The system is designed as a **frontend MVP** for interview and learning purposes, with a strong emphasis on clean architecture, modular Angular design, and reactive programming using RxJS.
